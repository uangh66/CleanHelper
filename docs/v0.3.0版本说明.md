# CleanHelper v1.0.0 版本说明

## 版本信息

- **版本号**：v1.0.0
- **发布日期**：2025-06-25
- **定位**：Windows 本地桌面清理工具 · 开源学习作品

## 当前功能

### 1. 临时文件清理

- 扫描当前用户 `%TEMP%` 目录
- 风险分级：按最后修改时间（>7天=低风险默认勾选，≤7天=中风险默认不勾选）
- 跳过 ReparsePoint（Junction / SymbolicLink）
- 只删除用户明确勾选且 `CanClean == true` 的文件
- 删除前对每个文件做路径安全二次校验
- 统计区分"成功处理大小"和"真实释放空间"
- 生成清理报告

### 2. 空间体检

- 扫描当前用户 6 个目录：下载、桌面、文档、图片、视频、音乐
- 可配置最小文件大小阈值：50MB / 100MB / 500MB / 1GB
- 仅读取文件元数据（路径、大小、修改时间、扩展名），不读取文件内容
- 按扩展名给出建议说明（压缩包/安装包/视频/镜像/其他）
- 不删除、不移动、不修改任何用户文件
- 支持"打开所在位置"（explorer.exe 定位）
- 生成空间体检报告

### 3. 聊天缓存分析

- 扫描微信、QQ、企业微信、钉钉的已知缓存目录（12个候选目录）
- 递归统计目录大小和文件数量
- 仅读取文件系统元数据，不读取聊天内容，不打开图片/视频/文档
- 按目录大小给出建议（<1GB / 1-5GB / >5GB）
- 目录不存在时显示"未发现"
- 支持"打开目录"（explorer.exe 打开）
- 生成聊天缓存分析报告

### 4. 工具功能

- 关于与安全说明弹窗
- 打开报告目录（logs 文件夹）
- 三种报告导出（清理报告 / 空间体检报告 / 聊天缓存报告）

## 安全边界

- 不联网，不上传文件
- 不后台常驻，不自启动，不安装服务
- 不清理注册表
- 不清理 Windows 系统关键目录
- 不清理 Program Files / Program Files (x86)
- 不清理用户桌面、文档、图片、视频（仅扫描展示）
- 聊天缓存分析不读取聊天内容
- 空间体检不读取文件内容
- 不做远程连接（WMI / WinRM / UNC）
- 不保存用户隐私数据到外部位置
- 扫描时跳过 ReparsePoint（Junction / SymbolicLink）
- 所有删除操作都有路径安全校验和二次确认

## 已知限制

1. 仅扫描当前登录用户的目录，不支持切换用户
2. 不支持命令行/静默模式，仅 GUI 操作
3. 不支持导出 Excel/PDF 格式报告，仅 TXT
4. 不支持自定义风险分级规则
5. 不支持扫描 OneDrive / 云同步目录（被 ReparsePoint 跳过）
6. 聊天缓存仅覆盖微信/QQ/企业微信/钉钉，不覆盖其他 IM 软件
7. 空间体检仅覆盖 6 个用户目录，不覆盖用户自建目录
8. 不支持自动更新

## 不适合的场景

- 未安装 .NET 8 Desktop Runtime 的环境
- 需要批量管理多台电脑的场景
- 需要清理注册表或系统目录的场景
